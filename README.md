# MADRE: Methodology for Network Data Analysis

Practical implementations of Machine Learning and Data Analysis techniques for computer networks.

## 🚀 Overview

This repository explores how data science can be applied to understand, evaluate, and secure computer networks.

## 📂 Modules & Practical Work (TPs)

The course is structured into several modules, each focusing on a specific class of algorithms:

1.  **Linear Regression** : Modeling relationships in network data.
2.  **Cross-Validation & Model Selection** : Evaluating model robustness and performance.
3.  **Classification** : Categorizing network traffic and identifying behaviors.
4.  **Clustering** : Unsupervised discovery of patterns and anomalies in network flows.
5.  **PCA & Anomaly Detection** : Dimensionality reduction and detecting unusual network events.
6.  **Neural Networks** : Deep learning for complex network pattern recognition.

## 🛠 Technologies

- **Language:** Python
- **Libraries:** NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn, PyTorch/TensorFlow (for NNs)
